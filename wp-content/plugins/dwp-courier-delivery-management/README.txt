Delivery management plugins for moving companies. Easy to create merchant account and delivery list in dashboard.

== Description ==

DWP Courier & Delivery Management is a delivery management wordpress plugins that you can use on any logistics website. Using this plugin, it's easy to create delivery request, merchant account and maintain all the features. Merchant details are auto detectable. So, they need to fill out the delivery information only. All details can be found in the admin dashboard and process the delivery as well.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `dwp-courier-management.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use dashboard option to maintain delivery and moving services.
